La usibilidad web

¿Qué es la usibilidad web?

La usibilidad web se podría definir en simples palabras en las características que hacen de una página
web sea sencilla de usar para las personas que la visitan, donde este busca que elementos como su diseño,
como se presenta al usuario, la organización de la información y su funcionamiento sea lo más amigable posible con el usuario.
Maria C.

¿Porqué es importante?

La importancia de la usabilidad en un sitio web es crucial para esta ya que si este es muy complicada de utilizar, su información no sea accesible o incluso
si esta es difícil de navegar provocando que muy probablemente la persona abandone la página para buscar otra mas fácil de utilizar.Por esto es muy importante la usabilidad
en un sitio web ya que sería mucho más probable que el usuario se quedara en la página si esta fuera mas ordenada y con su información a la mano facilmente para el usuario.
Sebastian G.

Principios de la usabilidad web

Entre los diez principios que existen de la usabilidad web encuentro más importantes los siguientes:

  1. Visibilidad del estado del sistema:
    Este principio nos quiere decir que si el usuario realizó alguna acción este deba recibir algún tipo de respuesta visual para indicarle si lo que hizo tendrá una respuesta de la página.
  2. Similitud con el mundo real:
    Aquí lo que se quiere dar a ver es que la página debe tener un lenguaje sencillo por así decirlo, que sus palabras sean fáciles de leer pero también los simbolos o imagenes sean fáciles de entender.
  3. Libertad de acción
    Lo más destacable de aquí es que es importante que la página le de el control al usuario de sus propias acciones además de controlar el sistema
  4. Estandares:
    Son ciertos modelos o diseños que se utilizan en páginas especificas las cuales el usuario ya está acostumbrado a ver en cierta estructura
  5. Prevencion de errores:
    La prevension de los errores es importante en la mayoria de los casos porque ayuda a prevenir que ciertas acciones del propio usuario provoque un mal funcionamiento de la página y por ende entorpecer la experiencia del propio usuario

Elena G. C.

Principios
Principio Ejemplo
Visibilidad del estado del sistema Una ventana de éxito para notificar al usuario
Similitud del sistema con el mundo real Un icono de una X para salir o cancelar
Libertad de acción Permitir que el usuario deshaga una acción con un botón o gesto
Estándares Colocar en una pagina web de compras un icono de carrito de compras
Prevención de errores Un mensaje de error y un icono con un signo de exclamación indicando que lo que se hará podría producir un error
